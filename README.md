# Digital Thermometer using Arduino

## Description
This project is a digital thermometer built using Arduino Uno and the LM35 temperature sensor. The temperature is measured in Celsius and displayed on a 16x2 LCD. The circuit was designed and tested using Tinkercad.

## Components Used
- Arduino Uno
- LM35 Temperature Sensor
- 16x2 LCD Display
- Potentiometer
- Breadboard
- Jumper Wires

## Software
- Arduino IDE
- Tinkercad

## Working
The LM35 sensor measures the ambient temperature and sends an analog signal to the Arduino. The Arduino converts the analog value into Celsius and displays the temperature on the LCD as well as the Serial Monitor.

## Output
Displays real-time temperature in °C.
